# 測試不同IP的網路延遲  
如果遊戲運行不順暢可以使用其他IP，來改善遊戲卡頓的問題。  
方法如下  
官方共提供4種不同的連線的IP  
<img width="199" height="444" alt="image" src="https://github.com/user-attachments/assets/4b4d41ce-23c5-40dd-8998-6dcd25622d3b" />  

可在`命令提示字元`中ping看看4種IP，選一組連線時間最短。
```
ping 121.9.243.212
ping 114.132.58.132
ping 134.175.217.176
ping 119.28.137.173

```
