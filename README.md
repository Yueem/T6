:bell:左右軟體開發的社群養成遊戲《第6大陸》又稱This6(玩家簡稱T6)，是由廣州左右軟體技術有限公司開發並經營的網頁遊戲。它是一款專為學生、上班族設計的FLASH虛擬互動社區，強調由自己去創造並享受生活。  
:exclamation:因遊戲開發較早，使用了已被移除的flash插件。導致海外玩家都無法下載與更新flash版本。  
:file_folder:本儲存庫提供**flash插件**，與推薦的**遊戲環境**，以及在遊玩過程中可能造成的**遊戲卡頓**問題。

# 測試不同IP的網路延遲  
如果遊戲運行不順暢可以使用其他IP，來改善遊戲卡頓的問題。  
方法如下  
官方共提供4種不同的連線的IP  
<img width="199" height="444" alt="image" src="https://github.com/user-attachments/assets/4b4d41ce-23c5-40dd-8998-6dcd25622d3b" />  

可在`命令提示字元`中ping看看4種IP，選一組連線時間最短。
```
ping 121.9.243.212
ping 114.132.58.132
ping 134.175.217.176
ping 119.28.137.173
```
